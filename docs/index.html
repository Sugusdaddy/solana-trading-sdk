Quick Examples

Simple Swap

const client = new TradingClient(connection, wallet);
await client.swap({
  inputMint: 'SOL',
  outputMint: 'USDC',
  amount: 1.5,
  slippageBps: 50
});

DCA Order

const dca = new DCAStrategy(connection, wallet);
await dca.createOrder({
  inputMint: USDC,
  outputMint: SOL,
  totalAmount: 1000,
  intervals: 10,
  intervalMs: 3600000 // hourly
});

Position with TP/SL

const position = await manager.openPosition({
  mint: tokenMint,
  amount: 100,
  entryPrice: 1.50,
  takeProfit: 2.00,  // +33%
  stopLoss: 1.20,    // -20%
});
